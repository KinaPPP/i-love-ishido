GAME MODES

ゲームモード

MODE A
[A] ALL WAYS MODE
The classic rules. Place a stone next to any stone that shares either the same color or the same number. Aim to empty the bag (LEFT: 0) and reach the CONGRATULATIONS! screen.
本家に準拠したルール。隣の石と色か数字のどちらかが一致すれば置けます。袋を空にして（LEFT: 0）CONGRATULATIONS!を目指しましょう。
★ Recommended for beginners
★ 初めての方におすすめ
MODE I
[I] ISHIDO+ MODE HARD
Stricter XOR rule. A stone must match either color or number — not both, not neither. Identical stones cannot be placed next to each other.
より厳しいXORルール。隣の石と色か数字の「どちらか一方だけ」が一致しなければ置けません。完全一致（同色同数字）も不可です。
★★ For experienced players
★★ 上級者向け
MODE E
[E] ENDLESS ISHIDO++ NEW
ISHIDO+ rules with a twist: WAY matches cause stones to dissolve and return to the bag. Use JOKER stones to survive STALEMATE. How many LOOPs can you complete?
ISHIDO+のルールに加え、WAYが成立すると石が消えて袋へ戻ります。ジョーカー石でSTALEMATEを乗り越えながら、何LOOPできるか挑戦しましょう。
★★★ The endless challenge
★★★ 終わりなき挑戦

PLACEMENT RULES

石の置き方

  • Stones are drawn one at a time from the bag (shown as NEXT).
  • A stone can only be placed next to an existing stone — isolated placements are not allowed.
  • In ALL WAYS: each adjacent stone must share the same color or the same number.
  • In ISHIDO+ / ENDLESS: each adjacent stone must match on exactly one attribute — color XOR number.
  • Placing next to 2, 3, or 4 stones at once scores a 2WAY / 3WAY / 4WAY!
  • If no valid placement exists, the game ends with STALEMATE.
  • 袋から1枚ずつ石を引いて盤面に置いていきます（NEXTに表示）。
  • 石は必ず既存の石の隣に置く必要があります。孤立した場所には置けません。
  • ALL WAYS：隣の石と色か数字のどちらかが一致すれば置けます。
  • ISHIDO+ / ENDLESS：隣の石と色か数字の「どちらか一方だけ」が一致する必要があります。両方一致も不可です。
  • 一度に2・3・4個の石と隣接して置くと 2WAY / 3WAY / 4WAY 達成！
  • どこにも置けなくなると STALEMATE（手詰まり）でゲーム終了です。

The 6 initial stones are placed at the four corners and two center squares before play begins. Aim to build chains — positioning stones where they touch multiple others creates WAY combos.

ゲーム開始時、四隅と中央2マスに計6枚の石が配置されます。複数の石と隣接するように置くことでWAYコンボが狙えます。

ENDLESS MODE IN DETAIL

ENDLESSモード詳細

  • WAY match: The placed stone and all adjacent matching stones dissolve and enter the SUB bag.
  • LOOP: When the main bag empties, the SUB bag shuffles and becomes the new main bag. LOOP count increases by 1.
  • JOKER stone: A special stone from your hand. Can be placed anywhere adjacent to an existing stone, regardless of color or number.
  • JOKER does not return to the bag when consumed. It stays on the board until a WAY is scored around it.
  • Board Clear: All 72 owned stones are cleared — JOKER stones remaining on the board do not prevent this! A special bonus triggers.
  • WAY成立時：置いた石と隣接する石が全て消え、SUB袋に入ります。
  • LOOP：メイン袋が空になると、SUB袋がシャッフルされてメイン袋に昇格し、LOOPカウントが1増えます。
  • ジョーカー石：手元にある特別な石。色や数字に関係なく、既存の石の隣であればどこにでも置けます。
  • ジョーカー石は消費されると袋には戻りません。WAYが成立するまで盤面に残り続けます。
  • 全消し：72枚の持ち石が全て消えると特別ボーナス発動！ジョーカー石が盤面に残っていても全消しと見なされます。

JOKER supply

ジョーカーの補給

  • Game start: 1 JOKER
  • Each LOOP achieved: +2 JOKERs
  • Board Clear bonus: +1 JOKER
  • ゲーム開始時：1個
  • LOOP達成毎：+2個
  • 全消しボーナス時：+1個

JOKER — COLOR mode: Rainbow flowing panel. Treated as matching any color.

JOKER — NUMBER mode: Grey panel with "?". Treated as matching any number.

Once a mode is chosen, UNDO history is cleared. The path is committed.

ジョーカー — COLORモード：虹色のパネルが流れます。どんな色の石とも一致するとみなされます。

ジョーカー — NUMBERモード：「？」のグレーパネル。どんな数字の石とも一致するとみなされます。

一度モードを選択するとUNDO履歴が全て消えます。一度決めた道は突き進め！

THEMES & ACCESSIBILITY

テーマと色覚サポート

Press T at any time, tap the title on the start screen, or click the NEXT stone display during play to cycle through five visual themes. Themes change instantly without interrupting gameplay.

Tキーを押すか、スタート画面のタイトルをタップ、またはゲーム中のNEXT石表示をクリックすると、5種類のテーマをいつでも切り替えられます。ゲームを中断せず即座に反映されます。

THEME 1
DEFAULT
Classic grey palette. Numeric tiles.
グレー系パレット。数字コマ。
THEME 2
DEFAULT CUD
Same design, color-vision-friendly palette. Blue → orange, green → dark green, cyan → platinum.
同デザイン、色覚サポート配色。青→オレンジ、緑→深緑、水色→プラチナに変更。
THEME 3
KANJI
Warm wood palette. Oval jewel tiles with kanji numerals 一〜六.
温かみのある木調パレット。楕円型コマに漢数字一〜六。
THEME 4
KANJI CUD
Kanji design with color-vision-friendly palette.
漢字テーマの色覚サポート版。
THEME 5
SEA
Deep-sea blue palette. Coin-bag tiles with Roman numerals I–VI. CUD-friendly by design.
深海ブルーのパレット。巻貝袋型コマにローマ数字I〜VI。CUD対応設計。

About CUD (Color Universal Design): The CUD themes replace blue, bright green, and cyan — colors that can be hard to distinguish for people with P-type or D-type color vision (affecting roughly 5% of people). The replacements — orange, dark green, and platinum — spread across different hue families so each stone remains visually distinct.

CUD（カラーユニバーサルデザイン）について：CUDテーマでは、P型・D型色覚（人口の約5%）で識別しにくい青・明緑・水色を、オレンジ・深緑・プラチナに置き換えています。それぞれ異なる色相に分散させることで、どのコマも視覚的に区別しやすくなります。

ABOUT THE ORIGINAL ISHIDO

オリジナル「石道」について

Ishido: The Way of Stones is a classic puzzle game first released in 1990 by Publishing International for the Macintosh. It was subsequently ported to DOS, Amiga, Atari ST, and various game consoles. The PC-98 version — the one KINA played — was published in Japan by ASCII Corporation, and left a lasting impression with its meditative pace and elegant stone-matching mechanics.

石道（Ishido: The Way of Stones）は、1990年にPublishing InternationalよりMacintosh向けに発売されたパズルゲームです。その後DOS・Amiga・Atari STや各種コンソール機へ移植されました。KINAが遊んだPC-9801版は、当時の株式会社アスキーから発売されており、瞑想的なテンポと美しい石のマッチング感覚が今でも忘れられません。

The core rule — place a stone that shares either the color or the number of each adjacent stone — sounds simple, but creates a surprisingly deep experience. Building toward a 4WAY feels as satisfying now as it did decades ago.

「隣の石と色か数字のどちらかが一致すれば置ける」というシンプルなルールながら、奥深い体験を生み出します。4WAYを狙う瞬間の緊張感は、数十年経った今でも変わりません。

I LOVE ISHIDO was built as a loving reimagination of that classic, adding ISHIDO+ mode with stricter XOR rules and the original ENDLESS ISHIDO++ mode with the JOKER stone mechanic.

I LOVE ISHIDOはそのオリジナルへのリスペクトから生まれた作品です。より厳しいXORルールのISHIDO+モードと、ジョーカー石を使ったオリジナルのENDLESS ISHIDO++モードを追加しました。

To the creators of the original Ishido — thank you for the stones, the paths, and the way.

オリジナル石道の制作者の皆様へ——石と道と、その先にある何かをありがとうございます。
