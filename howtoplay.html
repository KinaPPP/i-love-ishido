GAME MODES

MODE A
[A] ALL WAYS MODE
The classic rules. Place a stone next to any stone that shares either the same color or the same number.

Aim to empty the bag (LEFT: 0) and reach the CONGRATULATIONS! screen.
★ Recommended for beginners
MODE I
[I] ISHIDO+ MODE HARD
Stricter XOR rule. A stone must match either color or number with each adjacent stone — not both, not neither.

Identical stones cannot be placed next to each other. Reaching LEFT: 0 is a true achievement.
★★ For experienced players
MODE E
[E] ENDLESS ISHIDO++ NEW
ISHIDO+ rules with a twist: when you achieve a WAY match, all stones involved dissolve and return to the bag — keeping the board alive.

How many LOOPs can you complete?
★★★ The endless challenge
2

PLACEMENT RULES

  • Stones are drawn one at a time from the bag (shown as NEXT).
  • A stone can only be placed next to an existing stone — isolated placements are not allowed.
  • In ALL WAYS: each adjacent stone must share the same color or the same number.
  • In ISHIDO+ / ENDLESS: each adjacent stone must match on exactly one attribute — color XOR number.
  • Placing next to 2, 3, or 4 stones at once scores a 2WAY / 3WAY / 4WAY!
  • If no valid placement exists, the game ends with STALEMATE.

WAY scoring
The 6 initial stones are placed at the four corners and two center squares before play begins.
Aim to build chains — positioning stones where they touch multiple others creates WAY combos.

3

ENDLESS MODE IN DETAIL

  • WAY match: The placed stone and all adjacent matching stones dissolve and enter the SUB bag.
  • LOOP: When the main bag empties, the SUB bag shuffles and becomes the new main bag. LOOP count increases by 1.
  • JOKER stone: A special stone that can be placed anywhere adjacent. Use it to break through STALEMATE.
  • JOKER is consumed on use and does not return to the bag. The board placement remains until a WAY is scored.
  • Board Clear: If all stones dissolve from the board, a special bonus triggers — four corner stones are re-placed, and a bonus JOKER is awarded.

JOKER supply

  • Game start: 1 JOKER (a gift for the early game)
  • Each LOOP achieved: +2 JOKERs
  • Board Clear bonus: +1 JOKER

JOKER — COLOR mode
The JOKER stone displays a flowing rainbow panel. In COLOR mode, it is treated as matching any color.

JOKER — NUMBER mode
The JOKER stone displays a grey panel with "?". In NUMBER mode, it is treated as matching any number.

Once a JOKER mode is chosen, UNDO history is cleared — the path is committed.

SCREEN GUIDE

NEXT
The stone you are about to place. Click or press SPACE to place it on the highlighted cell.
LEFT
Stones remaining in the main bag. Reach 0 to win (ALL WAYS / ISHIDO+ modes).
SUB (ENDLESS)
Dissolved stones waiting in the secondary bag. When the main bag empties, SUB becomes the new bag.
LOOP (ENDLESS)
How many times the main bag has been cycled through.
JOKER (ENDLESS)
Number of JOKER stones in hand. Shown in red when available. Press [J] to use.
CURSOR
White border = valid placement. Red border = invalid. Move with mouse.

BUTTONS (bottom row)

[J] — JOKER
Opens the JOKER selection panel. Choose COLOR or NUMBER mode, or press [B] to cancel. Available in ENDLESS mode only.
[U] — UNDO
Reverts the last stone placement. Not available after using a JOKER.
[H] — HINT
Toggles the spiritual hint overlay — glowing dots show all valid placements.
[R] — RELOAD
Restart the current mode. A confirmation dialog appears before restarting.
[✕] — TITLE
Return to the title screen. A confirmation dialog appears. Located in the top-right corner.

ABOUT THE ORIGINAL ISHIDO

Ishido: The Way of Stones is a classic puzzle game first released in 1990 by Publishing International for the Macintosh. It was subsequently ported to various platforms including DOS, Amiga, Atari ST, and game consoles. The PC-98 version — the one KINA played — was published in Japan by ASCII Corporation, and left a lasting impression with its meditative pace and elegant stone-matching mechanics.

The core rule — place a stone that shares either the color or the number of each adjacent stone — sounds simple, but creates a surprisingly deep experience. Building toward a 4WAY feels as satisfying now as it did decades ago.

I LOVE ISHIDO was built as a loving reimagination of that classic, adding ISHIDO+ mode with stricter XOR rules and the original ENDLESS ISHIDO++ mode with the JOKER stone mechanic. The result is a game that honors the original while finding new depths in the same stone-on-board philosophy.

To the creators of the original Ishido — thank you for the stones, the paths, and the way.
